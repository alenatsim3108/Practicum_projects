# Рачет, визуализация и анализ метрик сервиса доставки еды
___

Сервису доставки еды «Всё.из.кафе» нужна срочная аналитическая помощь.
Задачи связаны с подготовкой данных и визуализацией ключевых метрик для бизнеса. 

Необходимо было рассчитать метрики и разработать дашборд, который с разных сторон отразит состояние клиентской базы в городе Саранске. 

На основе этого дашборда составлен аналитический отчёт.
